# Blockchain

## Blockchain Implementation.
This is a repo of implementing blockchain in Racket using functional paradigms.

## Usage
`git clone https://github.com/khagankhan/blockchain.git`
`cd blockchain`
`raco make main.rkt`
`racket main.rkt`
